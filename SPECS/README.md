# Especificações do Farmstok AI

Esta pasta contém o contexto técnico e funcional do projeto Farmstok AI.

O código-fonte está localizado em `src/`. Os arquivos desta pasta não fazem parte da aplicação em produção. Eles servem como documentação e contexto para desenvolvedores e agentes de programação.

## Ordem recomendada de leitura

1. `PROJECT_CONTEXT.md`
2. `ARCHITECTURE.md`
3. `CODING_RULES.md`
4. `API_CONTRACTS.md`
5. `DATA_MODEL.md`
6. `CURRENT_STATE.md`
7. Especificação da funcionalidade em `features/`

## Regras

- O código deve respeitar estas especificações.
- Mudanças arquiteturais devem ser documentadas.
- O agente não deve alterar requisitos por conta própria.
- Em caso de conflito, a especificação mais específica e recente prevalece.
- O agente deve analisar os arquivos existentes antes de editar.
